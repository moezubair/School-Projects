Implementing a simple file system that can :
	Get the info from a .IMA file
	Retrieve files from .IMA file
	Add files to .IMA file

This project consists of three files:
disklist 
diskinfo
diskget


----------------------------------------------------------------------------

Formula used to convert two(2) 8-bits entries to one(1) 16-bit entry:
	= (most-significant-bit * 256)+(least-significant-bit)

NOTE: LITTLE ENDIAN STORES THE LEAST SIG FIG IN LOWER ADDRESS
